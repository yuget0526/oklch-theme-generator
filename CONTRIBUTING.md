# Contributing to a11yPalette

Thank you for your interest in contributing! We appreciate all efforts to make this tool better.

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/oklch-theme-generator.git
   cd oklch-theme-generator
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Create a branch** for your feature or fix:
   ```bash
   git checkout -b feature/amazing-feature
   ```
5. **Make your changes**.
6. **Commit your changes**:
   ```bash
   git commit -m "Add amazing feature"
   ```
7. **Push to your fork**:
   ```bash
   git push origin feature/amazing-feature
   ```
8. **Open a Pull Request** against the `main` branch.

## Development

- Run dev server: `npm run dev`
- Build for production: `npm run build`
- Lint code: `npm run lint`

## Code Style

- We use Prettier and ESLint.
- Please ensure your code is consistent with the existing style.
